# Free QR Code Generator API

Free QR code generator API - URL/WiFi/vCard/email/SMS QR codes as SVG or PNG. No key.

**Live tool & docs:** https://qr.wrapper-agency.com

## Quickstart - free API, no key required

```bash
curl "https://qr.wrapper-agency.com/api/v1/qr?type=url&data=https://example.com&format=svg"
```

Clean JSON, open CORS, no signup. Try the interactive tool and full docs at https://qr.wrapper-agency.com
